# Window Icon

Place your icon file at:

`src/main/resources/com/example/dames/icon.png`

The app loads this file automatically for the window/taskbar icon.

## Notes

- Recommended format: PNG
- Recommended sizes: 16x16, 32x32, 64x64, 128x128
- To use another file name or extension, update the resource path in `Main.java`
